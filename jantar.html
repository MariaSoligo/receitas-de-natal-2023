Sapicão
De
Natal



INGREDIENTES

✓ 2kg de file de peito desfiado
✓ 100g de passas
✓ 3 tomates cozidos
✓ suco de 1 limão
✓ 1 cebola pequena
✓ 700g de presunto
✓ 100g de azeite
✓ 200g de milho verde
✓ 1 colher de queijo parmesão
✓ 2 pimentões verdes
✓ 250g de maionese





MODO DE PREPARO

✓ Deixe a cebola em uma tigela com açucar até sair a acidez

✓ Junte todos os ingredientes em um recipiente grande.

✓ Coloque na geladeira por 60 minutos.

✓ Sirva gelado com batata palha e um ramo de sasinha.
